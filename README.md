## üìö Getting Started quickly

1. **Install requirements**:  
   Begin by `pip/pip3 install -r requirements`

3. **Set up Tensorflow GPU(Optional)**  
   Follow the [Official Documentation](https://www.tensorflow.org/guide/gpu?hl=zh-cn) to configure GPU Support

4. **Start FLASK App**  
  Start the app by `python3 app.py`
  
5. **Use the AutoGemUI**  
   Use browser [http://127.0.0.1:5000](http://127.0.0.1:5000)
